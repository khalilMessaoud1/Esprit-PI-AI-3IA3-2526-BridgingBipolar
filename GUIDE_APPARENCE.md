# 🎯 Guide d'Utilisation - Modifications Visuelles

## ✨ Nouvelles Fonctionnalités

### 📍 Accès aux Paramètres
1. Connectez-vous à votre compte
2. Allez à **Settings** (⚙️)
3. Cliquez sur **Appearance** (le nouveau tab)

### 🎨 Options Disponibles

#### 1. Mode Jour/Nuit (Day/Night Mode) ☀️🌙
**Boutons disponibles:**
- ☀️ Jour (Light) - Défaut, mode clair avec couleurs vives
- 🌙 Nuit (Dark) - Fond sombre, texte clair, couleurs adaptées

**Effet:** Change immédiatement la couleur de tout le site

---

#### 2. Taille de Police (Font Size) 📏
**4 options:**
- **Petit** (Small) - Pour ceux qui préfèrent une interface compacte
- **Normal** (Normal) - Défaut, taille standard
- **Grand** (Large) - Pour meilleure lisibilité
- **Très Grand** (Extra-Large) - Pour plus de clarté

**Effet:** Augmente la taille de TOUT le texte et des emojis du site

---

#### 3. Mode Daltonien (Colorblind Mode) 🎨
**Important:** Pour les personnes atteintes de daltonisme

**Modes disponibles:**

| Mode | Description | Pour qui? |
|------|-------------|-----------|
| **Normal** | Aucun filtre, couleurs normales | Personnes sans daltonisme |
| **Deutéranopie** | Faiblesse du vert | ~1% des hommes |
| **Protanopie** | Faiblesse du rouge | ~1% des hommes |
| **Tritanopie** | Faiblesse bleu-jaune | 0.001% de la population |

**Effet:** Adapte les couleurs des indicateurs de phase (emojis: 🟢 🟡 🟠 🔴 et couleurs) pour être distinctes même sans pouvoir différencier certaines couleurs.

---

#### 4. Luminosité (Brightness) 💡
**Curseur:** 60% à 100%
- Utile pour:
  - Écrans très lumineux (↓ baisse)
  - Environnements sombres (↑ augmente)
  - Sensibilité aux lumières

---

#### 5. Aperçu des Emojis (Emoji Preview) 👀
**Affiche:** 😞 😐 😄
- Change en temps réel avec la taille de police
- Montre comment les emojis du site s'agrandissent

---

## 🚀 Exemples d'Utilisation

### Exemple 1: Lecture Nocturne
1. Allez à Appearance
2. Activez **Mode Nuit** (🌙)
3. Ajustez luminosité à **70%**
4. Sélectionnez taille **Grand** pour meilleure lisibilité
5. ✅ Prêt pour lire vos données la nuit!

### Exemple 2: Daltonien Rouge (Protanopia)
1. Allez à Appearance
2. Sélectionnez **Protanopie** dans Mode Daltonien
3. Les indicateurs de santé changent de couleurs
4. Ils sont maintenant distincts pour votre type de vision!

### Exemple 3: Accessibilité Maximale
1. Taille Police: **Très Grand** (20px)
2. Mode Jour/Nuit: **Nuit** 🌙
3. Luminosité: **80%**
4. Mode Daltonien: **Votre type**
5. ✅ Interface complètement adaptée à vos besoins!

## 🐛 Problèmes Connus & Résolutions

### "Les changements ne s'appliquent pas"
**Solution:** Rafraîchissez la page (F5 ou Ctrl+R)

### "Le mode nuit est trop sombre"
**Solution:** Augmentez la luminosité à 80-90%

### "Les emojis ne s'agrandissent pas"
**Solution:** 
1. Changez la taille de police
2. Rafraîchissez (F5)
3. Vérifiez votre cache navigateur
